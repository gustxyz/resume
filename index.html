Experience

  • Harmonic Northwest

    Frontend Developer (Freelance Contract)

    Nov 2021 - Present (4+ months)

    My first freelancing project was in conjunction with Harmonic Northwest to build a serverless shopify store with custom branding for pool supply company in arizona called Shasta Pools

    Shasta was built statically using Next.js to pre-render over 200 product pages pulling directly from shopify's headless graphql api, and then serve them via a CDN as a static react bundle.

    This setup allowed us complete freedom to design the site without being constrained by Shopify's template system -- the static nature of the site made it secure and scalable by default.

    What I really loved about the Shasta project was being able to showcase things like search and customer checkout in a serverless environment.

    React | Next.js

    Shopify Storefront Graphql Api

Projects

Cloud Shepherd

Open Source | Multi-Cloud | Cloud to Cloud Migration |

A promise based, multi-cloud, POISIX inspired client wrapper that abstracts away the complexities of Object Storage services, such as AWS's s3, as well as File Storage services, such as Dropbox, in order to easily interact with and move files between multiple clouds using one intuitive, familiar, and easy to use interface.
